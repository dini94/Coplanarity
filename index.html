DSSC Dye Descriptor Toolkit

Coplanarity / Planarity Descriptor

Quantify how planar a dye's backbone is, two ways: a corrected dihedral-angle averaging formula (quick, from angles alone), or the literature-standard Molecular Planarity Parameter (MPP) & Span of Deviation from Plane (SDP) computed from a best-fit plane through the actual 3D geometry.

Each calculator in this toolkit is a single self-contained HTML file — no build step, no install, just open it. New descriptor categories are added incrementally as they're validated against real DFT/TD-DFT data.
